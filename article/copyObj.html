# 深拷贝和浅拷贝

# 名词解释

浅拷贝： 只复制一层对象，当对象的属性是引用类型时，实质复制的是其引用。当引用值指向发生变化时，也会跟着改变
深拷贝： 复制多层对象属性值，改变属性值时，不影响原对象内任何属性值

# 使用场景

都是复杂对象，即对象的属性还是对象

方式二：递归实现深拷贝

let deepClone = obj => {
  let newObj = Array.isArray(obj) ? [] : {};
  if (obj && typeof obj === 'object') {
    for (let key in obj) {
      if (obj.hasOwnProperty(key)) {
        if (obj[key] && typeof obj[key] === 'object') {
          newObj[key] = deepClone(obj[key]);
        } else {
          // 如果不是对象直接拷贝
          newObj[key] = obj[key];
        }
      }
    }
  }
  return newObj;
}
